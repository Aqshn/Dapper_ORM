# Dapper_ORM

**Dapper_ORM** is a C# project that demonstrates how to use the **Dapper** micro-ORM for fast and efficient data access.

Dapper is a lightweight object-relational mapper that extends ADO.NET and allows developers to execute SQL queries and map results to .NET objects easily and efficiently.:contentReference[oaicite:1]{index=1}
